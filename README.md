# 💼 Employee Performance Predictor using Machine Learning

## 📌 Project Overview

This project predicts employee performance (High / Medium / Low) using Machine Learning based on employee behavior and work metrics.

## 🎯 Problem Statement

Companies struggle to identify employee performance early. This system helps HR teams predict performance and take proactive decisions.

## 💡 Solution

A Random Forest ML model is trained on employee features such as:

* Attendance
* Training hours
* Projects completed
* Experience
* Salary

## 🧠 Machine Learning Model

* Algorithm: Random Forest Classifier
* Type: Multi-class classification
* Output: High / Medium / Low performance

## 📊 Features Used

* Age
* Experience
* Salary
* Training Hours
* Attendance
* Projects Completed

## 🏗️ Project Workflow

1. Data Generation (Synthetic HR dataset)
2. Data Analysis (EDA)
3. Model Training
4. Evaluation
5. Prediction System
6. Streamlit Web App

## 🌐 Web Application

Built using Streamlit for real-time prediction.

## ▶️ How to Run

```bash
pip install -r requirements.txt
python src/data_generation.py
python src/model_training.py
streamlit run app/app.py
```
